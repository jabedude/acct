# Rust acct - parse acct(5) files

[![Crates.io](https://img.shields.io/crates/v/acct.svg)](https://crates.io/crates/acct)

A library for handling UNIX process accounting files.

To install, add this line to your Cargo.toml:

```toml
[dependencies]
acct = "0.4.0"
```
